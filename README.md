## 🌟 The Challenge
Students, professionals, and entrepreneurs frequently miss deadlines and important commitments. Existing productivity tools rely on passive reminders that are incredibly easy to ignore. **The challenge was to build an AI companion that proactively forces meaningful action.**

## 💡 The Solution: Priora
Priora eliminates procrastination by acting as your autonomous strategist. It doesn't just store tasks—it **analyzes your workload, predicts impending risks, and structures a bespoke daily battle plan (Morning, Afternoon, Evening).**

To bridge the gap between planning and execution, Priora introduces **Focus Mode**: a full-screen, skeuomorphic environment powered by ambient YouTube soundscapes designed to induce deep work. Combined with a fully hands-free **Voice AI Assistant**, Priora builds the perfect environment for you to succeed.

## ✨ Key Features & Innovations

- 🧠 **Intelligent Task Prioritization & Predictive Risk Modeling:** Dynamic dashboard sorting and context-aware "Predicted Risks" sidebars that warn you *before* you fail.
- 🤖 **Autonomous AI Strategist:** Gemini 2.5 autonomously parses your entire backlog and structures a cognitively optimized timeline.
- 🎙️ **Voice-Enabled NLP Assistant:** Hands-free productivity using native browser Speech Recognition and TTS. Just speak, and the AI executes database mutations.
- 🎧 **Immersive Focus Mode:** Ultra-low bandwidth YouTube audio integrations (Interstellar, Avatar, Harry Potter) combined with personalized "Deep Work Suggestions".
- 📈 **Goal & Habit Tracking:** Beautiful dashboards for tracking long-term consistency.
- 📱 **Progressive Web App (PWA):** Fully installable on desktop and mobile for a native application experience.

## 🛠️ Built With

- **Frontend:** Next.js 14, React, Tailwind CSS, Framer Motion
- **Backend & Database:** Next.js Edge Routes, Supabase (PostgreSQL)
- **AI & Logic:** Google Gemini API (2.5 Flash & Pro)
- **Media Engine:** YouTube Embed API
- **Browser APIs:** Web Speech API

## 🚀 Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Set up your `.env.local` with your Supabase credentials and Gemini API Key.
4. Run the development server: `npm run dev`
5. Open `http://localhost:3000`
